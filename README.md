# OpenClaw App Store

Registry and index for OpenClaw skills and extensions.

**Browse:** [clawhub.ai](https://clawhub.ai)

## Installation

Browse skills at [clawhub.ai](https://clawhub.ai) then install via:

```bash
# Using clawhub CLI
clawhub install <skill-name>

# Or via OpenClaw directly
openclaw skills install <skill-name>
```

## Available Categories

- **AI & Agents** — autonomous task execution, proactive behavior
- **Productivity** — calendar, tasks, notes, memory
- **Smart Home** — Tado, Hue, Google Home integration
- **Web & Research** — search, scraping, academic research
- **Developer** — GitHub integration, deployment tools

## Development

```bash
pnpm install
pnpm build    # production build
pnpm dev      # development server
```
